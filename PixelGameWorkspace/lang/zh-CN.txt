# Sprite
SN_Sprite=精灵
ST_Sprite=单贴图(无光照)
ST_Sprite_Texture=展示贴图
ST_Sprite_Color=顶点色
ST_Sprite_Anchor=锚点

# LitSprite
SN_LitSprite=精灵(受光)
ST_LitSprite=单贴图(带光照计算)
ST_LitSprite_Albedo=反射率贴图
ST_LitSprite_Normal=法线贴图
ST_LitSprite_Specular=高光贴图
ST_LitSprite_LightingLayer=光照层(0 -> 3 依次从近到远)
ST_LitSprite_Color=顶点色
ST_LitSprite_Cutoff=Alpha裁剪(用于Alpha Test)
ST_LitSprite_Anchor=锚点

# PointLight
SN_PointLight=点光
ST_PointLight=点光源(球形放射且强度随距离衰减)
ST_PointLight_Icon=图标
ST_PointLight_Texture=光强贴图
ST_PointLight_LightingLayer=光照层(0 -> 3 依次从近到远)
ST_PointLight_Color=光源颜色
ST_PointLight_Radius=光源半径
ST_PointLight_Intensity=光照强度

# PlayerSpawn
SN_PlayerSpawn=玩家出生点
ST_PlayerSpawn=玩家的出生点，一般在切地图点的附近放置。
ST_PlayerSpawn_IsGameStart=是否是游戏刚开始时的出生点，如果配置有多个这类节点，游戏中只会取到第一个。

# Fore Tiled Platform
SN_ForeTiledPlatform=前景瓦片平台
ST_ForeTiledPlatform=前景瓦片平台
ST_ForeTiledPlatform_Texture=Tileset贴图
ST_ForeTiledPlatform_BrightnessNoise=控制随机斑纹的噪声图
ST_ForeTiledPlatform_Color=顶点色
ST_ForeTiledPlatform_Cutoff=Alpha裁剪(用于Alpha Test)
ST_ForeTiledPlatform_LuminanceThreshold=亮度阈值，低于该阈值的颜色不会参与混合。
ST_ForeTiledPlatform_BrightnessScale=噪声图采样Scale
ST_ForeTiledPlatform_BrightnessStrength=斑纹强度
ST_ForeTiledPlatform_BrightnessQuantizationSteps=噪声点离散程度
ST_ForeTiledPlatform_BrightnessColor=斑纹颜色(用于叠加)

# Frame Animation
SN_FrameAnimation=帧动画
ST_FrameAnimation=帧动画
ST_FrameAnimation_Atlas=帧动画Sheets
ST_FrameAnimation_Prefix=图集的动画名前缀(数量需要和Atlas一致)
ST_FrameAnimation_Uniforms=每个图集对应的uniform名称(数量需要和Atlas一致)
ST_FrameAnimation_Interval=动画间隔
ST_FrameAnimation_FrameDescription=帧描述符，必须由逗号分隔，每个部分可以用F:N表示第F帧持续N帧，或者用N1-N2来表示连续播放[N1, N2]。示例：1,3:2,4-6,7-4,2:2,1
ST_FrameAnimation_Loop=是否循环
ST_FrameAnimation_Next=衔接的下一个动画，如果已经勾上循环则无效。

# Export Settings
ST_ExportSettings_DefaultLevelName=游戏刚开始时加载的关卡名称
ST_ExportSettings_GlobalUniforms=需要导出的全局Uniforms，必须在pipeline_hook中设置且类型必须是Float4。
ST_ExportSettings_ExceptLevels=不参与导出的关卡列表(输入名称即可)
